01 ABOUT ME TEXT

Personal Self-Introduction

My name is Nazwa Hayati. I was born in Kabupaten Semarang on September 15, 2009. I lived in Dusun Deresan, RT 03/04, Kecamatan Susukan. I was a student of class 10 PPLG 2 at SMK N TENGARAN. I chose the PPLG major because I was interested in technology, especially programming and website development.

At school, I learned about coding, website design, and computer technology. I enjoyed studying HTML and CSS because I could create creative website designs. One of my experiences was making a portfolio website project. Before coding, I designed the website in Figma and then continued the coding process step by step.

"In my free time, I enjoyed listening to music, learning new things about programming, and exploring design ideas. I was a hardworking, creative, and responsible person who always wanted to improve my skills and gain new experiences."
02 STORY OF CREATION

My Experience Creating a Portfolio Website Project

Phase 1: Designing UI

At the end of this semester, my classmates and I worked on a collaboration project to create a portfolio website. The first step was designing the website in Figma. I created several sections such as Home, About Me, Skills, Projects, and Contact. I also chose colors, images, and layouts to make the website look modern and attractive.

STEP 01

Phase 2: HTML & CSS Coding

After the design was approved, I started making the coding using HTML and CSS. I added navigation menus, buttons, images, and project cards. During the process, I faced several difficulties, such as layout errors and coding mistakes. Sometimes the layout did not match the design in Figma, and text or images looked messy on smaller screens.

STEP 02

Phase 3: Problem Solving

To solve the problems, I searched for tutorials on YouTube and Google. I also checked my code carefully and asked my teacher and friends for help. After practicing and fixing the errors many times, I finally succeeded in completing the website. This project gave me new experiences in designing, coding, teamwork, and problem-solving skills.
